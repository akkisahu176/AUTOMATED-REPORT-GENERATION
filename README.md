## 🛠️ Quick Installation

### Method 1: Quick Setup (Recommended)
```bash
# Clone the repository
git clone https://github.com/akkisahu176/AUTOMATED-REPORT-GENERATION.git

# Navigate to directory
cd AUTOMATED-REPORT-GENERATION

# Install dependencies
pip install -r requirements.txt

# Launch application
python data_analyzer_app.py
```

### Method 2: Manual Installation
```bash
# Install required packages individually
pip install pandas matplotlib seaborn customtkinter reportlab numpy

# Download and run
python data_analyzer_app.py
```

### 📋 System Requirements
- **Python**: 3.7 or higher
- **Memory**: 4GB RAM recommended
- **Storage**: 100MB free space
- **OS**: Windows 10+, macOS 10.14+, Ubuntu 18.04+

## 🐛 Troubleshooting

<details>
<summary><b>🚨 Common Issues & Solutions</b></summary>

### **File Upload Problems**
```bash
❌ Problem: "Cannot read file"
✅ Solution: Ensure file isn't open in Excel, check permissions

❌ Problem: "Unsupported format"  
✅ Solution: Use only .csv, .xlsx, or .xls files: Use only .csv files
```

### **Performance Issues**
```bash
❌ Problem: Slow loading with large files
✅ Solution: App automatically samples large datasets for preview

❌ Problem: Memory errors
✅ Solution: Close other applications, filter data to reduce size
```

### **Chart Display Issues**
```bash
❌ Problem: Charts not showing
✅ Solution: Ensure numeric columns exist, try different chart types

❌ Problem: PDF generation fails
✅ Solution: Check write permissions and disk space
```

</details>
